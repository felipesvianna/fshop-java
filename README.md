# Fshop API

### Tech Stack

- Sprint Boot Framework
- Mongo DB
- Mongo Express
- Docker

### How to start the API

1. Start the database
```
sudo docker-composer up 
```

2. Start the server
```
mvn spring-boot:run
```

URL to access Mongo Express

http://localhost:8081/

URL to access the API

http://localhost:8080/api/v1
